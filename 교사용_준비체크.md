# 교사용 준비 체크

## 전날

- [ ] GitHub Pages 또는 기관 서버에 WebAR 폴더를 올리고 HTTPS 주소가 열리는지 확인함.
- [ ] `AR_생태관찰_마커_A4.pdf`를 **100% 크기**로 출력함.
- [ ] 마커를 우드락/두꺼운 종이에 붙여 평평하게 준비함.
- [ ] 태블릿 1-2대에서 카메라 권한을 허용함.
- [ ] 교실 와이파이로 실제 주소가 열리는지 확인함.

## 수업 직전

- [ ] 마커의 검은 테두리와 흰 여백이 가려지지 않았는지 확인함.
- [ ] 마커를 창문 반사광이 강하지 않은 곳에 배치함.
- [ ] 재활용품 숲의 가장 높은 구조물이 태블릿 카메라에서 마커를 가리지 않게 조정함.
- [ ] 아이들이 동시에 태블릿을 잡지 않도록 관찰 순서를 정함.

## 인식이 잘 안 될 때

1. 태블릿을 마커에서 40-80cm 정도 떨어뜨립니다.
2. 마커가 화면에 네 모서리까지 다 들어오게 합니다.
3. 화면이 너무 어두우면 조명을 켜거나 자리만 옮깁니다.
4. 마커가 휘었으면 평평한 판에 다시 붙입니다.
5. 카메라가 열리지 않으면 주소가 `https://`인지, 브라우저 권한이 허용인지 확인합니다.
