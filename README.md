# 우리 숲에 찾아온 멸종위기동물 친구들 - WebAR 활동물

## 무엇이 들어 있나요

- `index.html`, `style.css`, `app.js`: 카메라로 마커를 비추면 숲 위에 동물이 나타나는 WebAR 앱
- `assets/forest-marker.patt`: 앱이 인식하는 맞춤형 AR 패턴 파일
- `assets/forest-marker.png`: 마커 원본 이미지
- `AR_생태관찰_마커_A4.pdf`: 바로 인쇄해 쓸 수 있는 A4 마커 카드
- `수업_운영안.md`: 유아 활동 흐름과 교사 발문
- `교사용_준비체크.md`: 설치/테스트 체크리스트

## 먼저 알아둘 점

이 활동물은 **마커 기반 WebAR**입니다. 순수 공간 인식 AR보다 교실 작품에 동물을 고정해 보이게 하기 쉽고, 별도 앱 설치 없이 브라우저에서 실행할 수 있습니다. 다만 **카메라 권한과 HTTPS 주소**가 필요합니다. `file://`로 파일을 직접 열면 카메라가 작동하지 않습니다.

## 가장 쉬운 실행 방법 - GitHub Pages

1. 이 폴더의 압축을 풉니다.
2. GitHub에서 새 저장소를 만듭니다. 예: `ar-forest`
3. 압축을 푼 폴더 안의 모든 파일을 저장소 최상단에 업로드합니다.
4. 저장소의 **Settings - Pages**에서 `Deploy from a branch`를 고르고, `main / (root)`를 선택합니다.
5. Pages 주소가 생성되면 휴대폰 또는 태블릿에서 그 주소를 엽니다.
6. 카메라 사용을 허용하고, 출력한 `AR_생태관찰_마커_A4.pdf`의 검은 테두리 전체가 카메라에 들어오게 비춥니다.

## 숲 작품 배치 권장안

- 출력한 마커는 **A4 100% 크기**로 인쇄하고, 평평한 우드락이나 두꺼운 종이에 붙입니다.
- 마커는 숲 작품의 **앞쪽 중앙**에 놓습니다. 마커의 검은 테두리, 흰 여백에는 다른 그림이나 장식물을 덮지 않습니다.
- 재활용품 숲은 마커를 기준으로 좌우 약 15-20cm, 뒤쪽 약 20cm 정도까지 배치하면 가상 동물이 실제 작품 안에 있는 느낌을 만들기 좋습니다.
- 동물 AR 위치: 왼쪽 앞 `반달가슴곰`, 오른쪽 앞 `산양`, 오른쪽 물가 `수달`, 뒤쪽 물가 `저어새`.

## 실행 팁

- 태블릿은 마커에서 약 40-80cm 떨어뜨린 뒤 천천히 움직입니다.
- 검은 테두리가 흐리거나 가려지면 인식이 약해집니다.
- 반사되는 코팅지보다 일반 복사용지, 무광 코팅지, 우드락 부착이 안정적입니다.
- 교실 와이파이가 필요합니다. 앱은 A-Frame과 AR.js를 CDN에서 불러옵니다.
- 동물은 손가락으로 눌러 짧은 보호 메시지를 볼 수 있습니다. 기기에서 음성 출력이 켜져 있으면 메시지를 읽어 줍니다.

## 수업 전 3분 테스트

1. 배포된 웹 주소를 태블릿에서 열고 카메라를 허용합니다.
2. 출력한 마커를 책상 위에 둡니다.
3. 마커가 화면에 완전히 들어오게 비춰 숲과 동물이 나타나는지 확인합니다.
4. 반달가슴곰, 수달, 산양, 저어새 중 하나를 눌러 설명 카드가 뜨는지 확인합니다.

## 기술 구성

- A-Frame 1.6.0
- AR.js 3.4.8의 패턴 마커 기능
- 모든 동물과 숲 오브젝트는 외부 3D 모델 없이 A-Frame 기본 도형으로 구성했습니다. 따라서 대용량 모델을 내려받지 않아도 됩니다.

## 근거 확인 메모

환경부 자료에서 반달가슴곰, 수달, 산양, 저어새가 국내 멸종위기 야생생물 사례로 제시됩니다. 앱의 메시지는 유아용 보호 행동 대화에 맞춰 단순화했습니다.

- 환경부: DMZ 생물다양성 조사 자료
- AR.js 공식 문서: marker based AR / pattern marker 안내
- AR.js 공식 저장소: 3.4.8 배포 태그
