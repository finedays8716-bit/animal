# 멸종위기동물 AR 서식지 관찰 - 동물별 표찰 버전

## 구성
- 모바일/태블릿용 WebAR 앱
- 동물별 AR 표찰 4종
  - 반달가슴곰
  - 수달
  - 산양
  - 저어새
- A4 출력용 표찰 모음 PDF/PNG

## 사용 방법
1. 아이가 동물 한 마리를 고릅니다.
2. 그 동물의 서식지를 그림 또는 재활용품으로 꾸밉니다.
3. 해당 동물의 AR 표찰 카드를 작품 앞에 붙입니다.
4. 웹페이지를 열고 카메라를 허용합니다.
5. 작품에 붙인 표찰을 비추면 그 동물만 나타납니다.

## GitHub Pages 올리는 방법
- 이 폴더 안의 파일들을 저장소 첫 화면(root)에 올립니다.
- `index.html`, `style.css`, `app.js`, `assets` 폴더가 바로 보여야 합니다.
- HTTPS 주소에서 열어야 카메라 사용이 됩니다.

## 권장 환경
- 안드로이드 휴대폰 또는 갤럭시탭 + Chrome
- 밝은 곳, 평평한 표찰
