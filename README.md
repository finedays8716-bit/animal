# 우리 숲에 찾아온 멸종위기동물 친구들 — 공간 AR 선택형 수정본

## 이번 수정에서 바뀐 점
- 동물이 한 번에 모두 나오지 않고 **한 마리씩 선택**해서 배치합니다.
- 기존 마커 방식이 아니라 **공간 AR(WebXR hit-test)** 방식입니다.
- 카메라 화면은 기기 화면에 맞춰 **전체 화면**으로 실행되도록 구성했습니다.
- 동물 이미지는 이전보다 더 **실사에 가까운 컷아웃 이미지**로 교체했습니다.
- 동물을 놓은 뒤에는 같은 AR 세션 동안 화면을 움직여도 **그 자리에 계속 보이도록** 구성했습니다.
- `다시 놓기`, `동물 바꾸기` 버튼을 추가했습니다.

## 사용 방법
1. GitHub 저장소 최상단(root)에 이 폴더 안의 파일을 그대로 올립니다.
2. GitHub Pages를 켭니다.
3. 배포 주소를 **안드로이드 Chrome**에서 엽니다.
4. 동물 한 마리를 고릅니다.
5. `선택한 동물과 AR 시작하기`를 누릅니다.
6. 숲 작품의 바닥이나 받침대를 천천히 비춥니다.
7. 발자국 표시가 보이면 `여기에 놓기`를 누르거나 화면을 한 번 눌러 동물을 놓습니다.

## 중요 안내
- 이 버전은 **공간 AR(WebXR immersive-ar)** 이므로, iPhone/iPad Safari나 일부 브라우저에서는 실행되지 않을 수 있습니다.
- 가장 안정적인 환경은 **ARCore 지원 안드로이드 기기 + Chrome** 입니다.
- 공간 인식이 잘 안 되면 작품 받침대나 바닥에 **무늬가 있는 표면**, 충분한 **밝기**가 필요합니다.
